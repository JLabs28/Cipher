How to Play - Cipher Hunt

Learn how to encrypt and decrypt messages using the Caesar Cipher!

What is the Caesar Cipher?

The Caesar Cipher is a simple encryption technique in which each letter of the plaintext is replaced by a letter with a fixed number of positions down or up the alphabet.

Steps to Play:

  1. Select whether you want to encrypt or decrypt a message.
  2. For encryption, input your message and choose a shift value (key). The shift value determines how many letters each character in your message will move. For example, a shift of 3 would change 'A' to 'D', 'B' to 'E', and so on.
  3. For decryption, input the encrypted message and provide the shift value. The system will reverse the process to retrieve the original message.
  4. Click "Submit" to see your encrypted or decrypted message!

Example:

Plaintext: HELLO

Shift Value: 3

Encrypted Text: KHOOR
